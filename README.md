# Learn Go

在使用 Go 去写一些小玩意时，我总感觉不是很顺畅。而且我写的代码有时更像 PHP 的风格，未能充分利用 Go 的特性。到后来我意识到，可能是因为跳过了基础的学习，经典的“Hello, World!”也没有好好完成。刚才我在搜索一个简单问题时，抬头看到书架上已经落灰的几本 Go 书时，我意识到我应该从零开始，以一个新手的姿态，重新学习 Go 语言，逐步提升。

所以，我想记录下在学习 Go 语言时的经验与积累。希望这些分享能够帮助其他想要入门 Go 的小伙伴们。
