# Learn Go

在使用 Go 编写一些小项目时，我总感觉不够顺畅，代码更像是用 Go 的语法重写了 PHP 的逻辑，没能充分利用 Go 的优势。后来我才意识到，可能是我跳过了基础的学习，甚至连经典的“Hello, World!”都没有认真完成。刚才在解决一个简单问题时，抬头看到书架上那些早已积灰的 Go 书，我才恍然大悟：是时候从头开始，像新手一样，重新学习 Go 语言，逐步提升自己了。

所以，我想记录下在学习 Go 语言时的经验与积累，希望这些分享能够帮助其他想要入门 Go 的小伙伴们！
