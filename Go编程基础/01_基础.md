## 基本类型

### 1. **布尔型 (`bool`)**

* 占用 1 字节
* 取值：`true` 或 `false`
* **零值**：`false`

### 2. **整型（`int` / `uint`）**

* `int` 和 `uint` 类型的大小与平台有关：

  * 32 位平台：`int` 是 32 位
  * 64 位平台：`int` 是 64 位
* **零值**：`0`

### 3. **具体大小的整型**

* `int8` / `uint8`：1 字节

  * 范围：`-128~127` / `0~255`
  * `byte` 是 `uint8` 的别名
  * **零值**：`0`
* `int16` / `uint16`：2 字节

  * 范围：`-32768~32767` / `0~65535`
  * **零值**：`0`
* `int32` / `uint32`：4 字节

  * 范围：`-2^31 ~ 2^31-1` / `0~2^32-1`
  * **零值**：`0`
* `int64` / `uint64`：8 字节

  * 范围：`-2^63 ~ 2^63-1` / `0~2^64-1`
  * **零值**：`0`

### 4. **浮点型（`float32` / `float64`）**

* `float32`：4 字节，精度 7 位
* `float64`：8 字节，精度 15 位
* **零值**：`0.0`

### 5. **复数（`complex64` / `complex128`）**

* `complex64`：8 字节（每个浮点数 4 字节）
* `complex128`：16 字节（每个浮点数 8 字节）
* **零值**：`0 + 0i`

### 6. **指针型（`uintptr`）**

* 足够保存指针的 32 位或 64 位整数
* **零值**：`nil`

### 7. **其他值类型**

* **数组（`array`）**：固定大小的元素集合

  * **零值**：数组中的每个元素都被初始化为其零值。例如，`int[3]` 类型的零值是 `[0, 0, 0]`。
* **结构体（`struct`）**：自定义数据类型

  * **零值**：结构体中所有字段都被初始化为其零值。例如，`struct { Name string }` 的零值是 `{ Name: "" }`。
* **字符串（`string`）**：不可变的字符序列

  * **零值**：`""`（空字符串）

### 8. **引用类型**

* **切片（`slice`）**：动态数组

  * **零值**：`nil`
* **映射（`map`）**：无序键值对集合

  * **零值**：`nil`
* **通道（`chan`）**：用于 goroutine 之间的通信

  * **零值**：`nil`

### 9. **接口类型（`interface`）**

* 用于表示一组方法的集合
* **零值**：`nil`

### 10. **函数类型（`func`）**

* 表示函数签名
* **零值**：`nil`

## 运算符

### 1. **算术运算符**

用于基本的数学计算：

* `+`：加法
* `-`：减法
* `*`：乘法
* `/`：除法（整型除法会截断结果）
* `%`：求余（取余数）

**例子**：

```go
a := 10
b := 3
fmt.Println(a + b)  // 输出 13
fmt.Println(a - b)  // 输出 7
fmt.Println(a * b)  // 输出 30
fmt.Println(a / b)  // 输出 3
fmt.Println(a % b)  // 输出 1
```

### 2. **关系运算符**

用于比较两个值之间的关系：

* `==`：等于
* `!=`：不等于
* `>`：大于
* `<`：小于
* `>=`：大于或等于
* `<=`：小于或等于

**例子**：

```go
a := 10
b := 5
fmt.Println(a == b)  // 输出 false
fmt.Println(a != b)  // 输出 true
fmt.Println(a > b)   // 输出 true
fmt.Println(a < b)   // 输出 false
fmt.Println(a >= b)  // 输出 true
fmt.Println(a <= b)  // 输出 false
```

### 3. **逻辑运算符**

用于组合条件表达式：

* `&&`：逻辑与（AND）
* `||`：逻辑或（OR）
* `!`：逻辑非（NOT）

**例子**：

```go
a := true
b := false
fmt.Println(a && b)  // 输出 false
fmt.Println(a || b)  // 输出 true
fmt.Println(!a)      // 输出 false
```

### 4. **位运算符**

用于整数的二进制位操作：

* `&`：按位与
* `|`：按位或
* `^`：按位异或
* `&^`：按位清除（A 和 B 对应位都为 1 时，A 结果为 0，B 结果为 1）
* `<<`：左移
* `>>`：右移

**例子**：

```go
a := 5  // 二进制：0101
b := 3  // 二进制：0011
fmt.Println(a & b)   // 输出 1  (0101 & 0011 = 0001)
fmt.Println(a | b)   // 输出 7  (0101 | 0011 = 0111)
fmt.Println(a ^ b)   // 输出 6  (0101 ^ 0011 = 0110)
fmt.Println(a &^ b)  // 输出 4  (0101 &^ 0011 = 0100)

fmt.Println(a << 1)  // 输出 10  (左移一位，0101 << 1 = 1010)
fmt.Println(a >> 1)  // 输出 2   (右移一位，0101 >> 1 = 0010)
```

### 5. **赋值运算符**

用于将一个值赋给变量，Go 支持不同形式的赋值运算符：

* `=`：普通赋值
* `+=`：加法赋值
* `-=`：减法赋值
* `*=`：乘法赋值
* `/=`：除法赋值
* `%=`：求余赋值
* `<<=`：左移赋值
* `>>=`：右移赋值
* `&=`：按位与赋值
* `|=`：按位或赋值
* `^=`：按位异或赋值
* `&^=`：按位清除赋值

**例子**：

```go
a := 5
a += 3  // a = a + 3
fmt.Println(a)  // 输出 8

a *= 2  // a = a * 2
fmt.Println(a)  // 输出 16

a >>= 2  // a = a >> 2 (右移 2 位)
fmt.Println(a)  // 输出 4
```

### 6. **其他运算符**

* **`++`**：自增运算符，增加 1
* **`--`**：自减运算符，减少 1
* **`:=`**：声明并赋值（简短赋值运算符）
* **`,`**：多重赋值，用于同时赋值多个变量

**例子**：

```go
a := 5
a++  // 自增
fmt.Println(a)  // 输出 6

b := 10
c := 20
b, c = c, b  // 多重赋值
fmt.Println(b, c)  // 输出 20 10
```

### 7. **条件运算符**

Go 没有三元运算符（`?:`），但可以使用 `if` 语句代替三元运算符的功能：

```go
a := 10
result := 0
if a > 5 {
    result = 1  // 设为 1，如果 a 大于 5
} else {
    result = 0  // 否则设为 0
}
fmt.Println(result)  // 输出 1
```
